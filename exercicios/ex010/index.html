Links

Definição

Links são caminhos ou interligações que podem funcionar como acesso rápido.

Links Externos

Links Externos são links que funcionam como âncora para levar um usuário para outra página web que não seja a sua. Exemplo:
Me siga no Youtube

Links Internos

Links Internos são usados geralmente para acessar páginas web no seu próprio local ou página web. Exemplo:
Ver meu site de Animes
