Listas Em Html

Listas Ordenadas

Listas Ordenadas são aquelas que possuem uma determinada ordem, que são ordenadas ou organizadas pelo o HTML 5. Eis o exemplo:

  1. Acordar
  2. Ir para a escoala
  3. Jogar GTA
  4. Comer pão

Listas Não Ordenadas

São aquelas que não são ordenadas. Exemplo:

  • Acordar
  • Ir para a escoala
  • Jogar GTA
  • Comer pão
